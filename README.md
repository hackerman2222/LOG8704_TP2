# TP2 - LOG8704

## Description
Il s’agit d’un projet développé dans le cadre du TP2 de LOG8704 et hébergé sur GitHub.

L’objectif de ce README est de fournir toutes les informations nécessaires pour **builder et tester l’application**, ainsi que des détails sur les appareils et plateformes utilisées.

---

## Contenu du dépôt
- **/Assets** : Contient tous les assets nécessaires (textures, modèles 3D, sons…).  
- **/Scripts** : Contient les scripts C# utilisés dans le projet.  
- **/Scenes** : Contient la scène principale à ouvrir pour tester le projet.  

---

## Scène à utiliser
Pour tester le projet, ouvrez la scène suivante dans Unity :  /Scenes/SampleScene.unity

---

## Configuration requise
- **Unity version** : Nous avons utilisé la version 6000.0.54f1.  
- **Plateformes testées** :  
  - Tester sur la plateforme android avec un Samsung Galaxy Z flip 6.

---

## Instructions pour lancer le projet
```bash
1. Clonez le dépôt :  
git clone https://github.com/hackerman2222/LOG8704_TP2

2. Ouvrez Unity Hub et ajoutez le projet.

3. Ouvrez la scène principale : Assets/Scenes/SampleScene.unity

4. Brancher votre téléphone android (Le téléphone doit avoir le mode développeur activé).

5. Changer de plateforme pour build sur Android: File -> Build Profiles -> Android -> Switch Platform 

6. Lancer l'application: Allez dans File -> Build Profiles -> Build And Run.

7. Suivez les instructions affichées dans l'interface pour placer des plantes dans votre environnement.
